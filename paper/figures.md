# 필요한 그림/표 목록

## 그림

### 그림 1. 시스템 전체 구조도
- **내용**: 2단계 분류 파이프라인 흐름도
- **구성**:
  ```
  마이크 입력 → [1단계: YAMNet 521개] → 직접 매핑 (5개)
                                      → 2차 분류기 (5개)
                                      → 무시 (나머지)
  → 서버 전송 → LED/진동 알림
  ```
- **기존 자료**: docs/system_architecture.png (있지만 2단계 구조 반영 필요)
- **상태**: 수정 필요 (현재 구조에 맞게 업데이트)

## 표

### 표 1. 카테고리별 분류 정확도
- **내용**: 11개 카테고리 Precision / Recall / F1-Score
- **기존 자료**: docs/confusion_matrix.png에서 수치 추출
- **상태**: confusion_matrix.png에서 수치 확인 필요

### 표 2. 2단계 분류 구조 카테고리 매핑
- **내용**: 직접 매핑 5개 + 2차 분류기 5개 정리
- **상태**: 작성 가능

### 표 3. 기존 솔루션 비교 (선택)
- **내용**: Google Sound Notifications / Earzz / 소리우산 / Hearo 비교
- **상태**: 분량에 따라 포함 여부 결정

## 그림/표 배치 계획 (3페이지 기준)

| 위치 | 항목 | 크기 |
|------|------|------|
| 1페이지 하단 or 2페이지 상단 | 그림 1. 시스템 구조도 | 1단 너비 |
| 2페이지 중간 | 표 2. 카테고리 매핑 | 1단 너비 |
| 2페이지 하단 or 3페이지 | 표 1. 분류 정확도 | 1단 너비 |
